## Kidney John

By Josh Lapham [josh@joshlapham.com]

License: [Beerware](https://en.wikipedia.org/wiki/Beerware)

### What This Is

An iOS app for animator [John Paine](http://kidneyjohn.com) that showcases his animations, comics and random drawings.
